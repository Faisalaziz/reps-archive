TimezoneEurope/Stockholm
DescriptionAt the event there will be talks about Media & Information Literacy, Digital Inclusion and Digital Reference in libraries. Workshop on Mozilla Web Literacy Map to show how it fits into the work carried through in Swedish libraries. Library world needs to adopt new methods for digital inclusion and new models for collaborations with the web. Web Literacy Map can be a useful to map the digital skills and figure out the main areas in which libraries needs to offer help and guidance to people.
