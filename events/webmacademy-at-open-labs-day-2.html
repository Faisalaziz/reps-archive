DescriptionAfter the success of the first Maker Party some of the attendees of that event had the idea of a series of Maker Parties in a more intensive manner, in order to be more fluent in web literacy. This is how the idea of Webmacademy was born. Webmacademy is a series of workshops that will teach the attendees code philosophy and web literacy. It is mainly focused on the usage of the Webmaker platform while still using the MDN resources as a reference for HTML, CSS and Javascript.
