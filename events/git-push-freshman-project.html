DescriptionThe Git Push Freshman project aims to train Mozilla Maseno club members the concepts of version control and a distributed approach in collaboration while developing software using Git and GitHub. This will help club members effectively keep a record of changes in their software source code, host the source code in a public or private repository and invite other contributors to modify the computer program improving its quality in the process.
