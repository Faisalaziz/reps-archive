DescriptionWebExtensions is a cross-browser system for developing browser add-ons. To a large extent, the system is compatible with the extension API supported by Google Chrome and Opera. Extensions written for these browsers will in most cases run in Mozilla Firefox or Microsoft Edge with just a few changes. The API is also fully compatible with multiprocess Firefox.
