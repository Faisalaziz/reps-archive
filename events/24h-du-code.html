DescriptionA 24 hours hackathon, teams working on three proposed subjects, one being an awesome open web app for Firefox (OS). Mozilla Reps attending will help for the Open Web Apps coding and will elect the best implementation for the Mozilla subject: GTFS application. This is a specification for public transportations schedules and stops. The goal is to provide an awesome app for FirefoxOS that will help people with their public transportation, and have some offline computing capability.
