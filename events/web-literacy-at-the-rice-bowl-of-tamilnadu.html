DescriptionThanjavur district is termed as Rice Bowl of Tamilnadu. It has a prominent position in Indian history. Webliteracy is a must for kids in remote villages of this place. We have a plan to visit as many schools as possible in remote villages of this historical district where we would teach the web to kids. This would be a great initiatiative, more like Kidzilla programme. A one month event aimed at huge awareness about the web and Mozilla Mission
