DescriptionCodepen meetups have been happened since 2014 around the world. In 2016 Brazil (and Latin America) received this meetup, which happened on October 22th at Fatec Sorocaba. The format of the event is based on Show & Tell with curated speakers and Dribbble + CodePen dojo cross-overs. In 2017 the event will happen in São Paulo on November 25th. We will present a Pen about WebVR/A-frame at the event to show how easy is to start using A-frame to build a VR scene.
