DescriptionA-Frame is a web framework for building virtual reality (VR) experiences. This Mozilla-supported framework is powerful, open source, and is easy to learn. In this talk, we will explore the intro to trends in Virtual Reality and how one can create VR scenes for Web with visual 3D inspector without any coding and then doing the same with coding. We will look into cool demos of VR scenes created by Mozilla VR community.
