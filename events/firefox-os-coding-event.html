DescriptionMozilla would like to welcome everyone who is interested in the Firefox OS software stack to come and learn what it takes to port Firefox OS to new phones or IoT boards. We have no set agenda for this session, but rather we hope you have ideas. For instance, do you want to run Firefox OS on your DragonBoard [1]? Do you want to get bluetooth working on your Fx0 [2]? How about running Firefox OS on your Fairphone [3]? Well, this is the event where you can learn how.
