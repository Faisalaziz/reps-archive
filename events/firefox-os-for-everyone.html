DescriptionFirefox OS is an ecosystem in itself. Everyone, not just developers, can contribute to it, and help the ecosystem grow. The aim of this event is to help participants learn about Firefox OS from a user point-of-view. It will enable participants to talk about Firefox OS to non-developers, by understanding the core concepts, and learning to explain them in simple words.
