DescriptionA Hackathon for bringing contributors to the Add-ons Community. Extensions for Firefox are built using the WebExtensions API, a cross-browser system for developing extensions. To a large extent, the system is compatible with the extension API supported by Google Chrome and Opera and the W3C Draft Community Group. Extensions written for these browsers will in most cases run in Firefox or Microsoft Edge with just a few changes. The API is also fully compatible with multiprocess Firefox.
