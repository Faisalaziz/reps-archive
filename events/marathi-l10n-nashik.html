DescriptionThis event will be the first ever Localization event in Nashik and will be focused on training the new and interested community members on how to translate in Marathi language. This event will help set up a Marathi localizers team. The added advantage of having this event now is that the Firefox OS is out recently, and it needs localization in Marathi as well as some other regional languages as well. So this event may come in handy as it will help Firefox OS become more user friendly.
