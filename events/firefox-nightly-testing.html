DescriptionEvery day, Mozilla developers write code that is merged into a common code repository (mozilla-central) and every day that code is compiled so as to create a pre-release version of Firefox based on this code for testing purposes - this is what we call a Nightly build. Mozilla Mombasa community is hosting testing session on the Firefox Nightly browser. We will look in to quality testing and automated testing.
