DescriptionPlease join us at Supermarkt on Saturday February 2nd, to learn, hack and celebrate the Firefox OS, Mozilla's open source operating system for the mobile web. If you're a web developer, you’ll be right at home coding apps in HTML5, JavaScript and CSS for the Firefox OS. We’ll help you with everything else, including how to access mobile Web APIs and how to use the Firefox OS Simulator in your desktop browser to view and test your mobile apps.
