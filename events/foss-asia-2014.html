DescriptionFOSSASIA 2014 is an exciting Open Technology event for developers, students, and start ups using open source to build up their businesses. FOSSASIA 2014 will take place for the first time in Cambodia. It will be held at Norton University in Phnom Penh from February 28 - March 2. FOSSASIA 2014 will focus on the use of open technologies in sustainable development in order to improve the lives and conditions of those living in poverty.
